Introduction
The Medical Store Management System is a database management system that helps manage the day-to-day operations of a medical store. It provides a simple and user-friendly interface for the store manager to manage inventory, sales, and purchases. The system also helps keep track of customer information, staff information, and supplier information.

Features
The Medical Store Management System includes the following features:

Add, edit and delete products
View product information, including price, quantity, and expiry date
Record sales and generate sales reports
Record purchases and generate purchase reports
Add, edit and delete customers
Add, edit and delete staff members
Add, edit and delete suppliers
Requirements
To run the Medical Store Management System, you will need the following:

MySQL database server
PHP
Apache server
Web browser
Installation
Clone the repository to your local machine
Import the database file (medical_store_management.sql) to your MySQL server
Change the database configuration in the config.php file to match your MySQL server credentials
Upload the entire project to your Apache server
Usage
Open a web browser and navigate to the URL where the project is installed
Use the login credentials to log in to the system
Use the system to manage inventory, sales, purchases, customers, staff members, and suppliers
